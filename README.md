⚔️ SPARTAN WAR COMMAND (WAR INTEL SYSTEM)


---

🧠 What this script is

Spartan War Command is a real-time tactical intelligence overlay for Torn ranked wars.

It runs inside the Torn faction page and acts as a live war dashboard (PDA-friendly) that:

Tracks enemy faction members automatically

Monitors hospital status and cooldown timers

Identifies who is ready or still recovering

Lets you mark priority targets

Allows instant “CALL” copy for faction coordination


It is designed as a lightweight command interface, not an automation bot.


---

🎯 What it is used for

This script is used to:

📊 Monitor enemy activity during ranked wars

⏱ Track hospital cooldown timers in real time

⭐ Mark high-value or priority enemies

📋 Copy quick war callouts into faction chat

📱 Operate smoothly on mobile (Torn PDA-style usage)



---

⚙️ Main Features

🪖 Enemy War Tracking

Automatically detects active ranked war

Loads enemy faction members



---

⏱ Hospital Timer System

Shows how long enemies remain in hospital

Updates dynamically from Torn API



---

⭐ Priority System

★ Button = mark/unmark enemy as priority

Priority targets appear in a separate section



---

📋 CALL System

CALL button copies formatted data to clipboard:


Player Name | M:SS

Used for fast faction coordination


---

🔍 Search Bar

Filters enemies by name in real time



---

📱 Compact Mode (Toggle Button ▣ / ▥)

Collapses panel into small floating HUD

Enlarges buttons for finger-friendly control

Saves screen space on PDA



---

🧲 Drag System (⋮⋮ Handle)

Move panel anywhere on screen

Position is saved automatically



---

🔘 Button Functions

⋮⋮ Drag Handle

Hold and drag to move the panel

Used for repositioning UI



---

▣ / ▥ Toggle Button

▣ = Compact mode (collapsed HUD)

▥ = Expanded mode (full interface)



---

★ Priority Button

Adds/removes enemy from priority list

Prioritized enemies appear at top section



---

CALL Button

Copies enemy info + cooldown timer

Format:


Name | M:SS

Used for:

faction chat coordination

attack timing calls



---

Search Bar

Filters enemies by name instantly

Case-insensitive matching

📦 How to install (Torn PDA / mobile)

Step 1 — Install userscript manager

You need one of:

✔ Tampermonkey (recommended)

✔ Violentmonkey

✔ Kiwi Browser extension (Android)



---

Step 2 — Create script

1. Open Tampermonkey


2. Tap “Create new script”


3. Delete default code


4. Paste full Spartan War Command script


5. Save




---

Step 3 — Open Torn

Go to:

https://www.torn.com/factions.php

Refresh the page.

The panel should appear automatically.


---

🔑 How to get your Torn API key

Step 1

Go to:

https://www.torn.com/preferences.php#tab=api

Step 2

Click Generate API Key

Copy your key



---

⚠️ Important notes

Do NOT share your API key publicly

If API key is wrong → no war data will load
Use Limited Key

Script only works during:

active ranked wars


It does NOT run in background after app is closed
